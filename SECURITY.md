# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public issue.**

Instead, please send an email to the maintainers or use GitHub's private vulnerability reporting feature:

1. Go to the [Security tab](https://github.com/mateimicu/monorepo-dependabot-config/security) of this repository
2. Click "Report a vulnerability"
3. Provide a description of the vulnerability and steps to reproduce

We will acknowledge your report within 72 hours and aim to provide a fix within a reasonable timeframe.
